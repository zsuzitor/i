
создаем папки. например
proj/src/components/
proj/dist/        <-- папка выходных файлов

в proj: npm init
ИЛИ можно скопировать файл package.json

добавить в раздел devDependencies
npm install --save-dev lib_name_1 lib_name_2 lib_name_3
добавить глобально
npm install -g lib_name_1 lib_name_2 lib_name_3

то что в package.json ->scripts
запускается через npm run script_name



npm link typescript -- используем глобальную версию typescript
npm install typescript - если надо использовать локальную версию вместо глобальной


npm rm {тут тоже самое что при install} - удаляет зависимость
